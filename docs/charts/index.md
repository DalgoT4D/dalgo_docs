# Charts

**Charts are the visual building blocks of every Dalgo dashboard and report.**

A chart draws from a single dataset (a table in your warehouse) and displays that data as a bar chart, line chart, pie chart, number, map, or table. Charts are created independently and then added to one or more dashboards.

## In this section

- [Creating a chart](./creating-a-chart.md) — how to build a new chart from a dataset
- [Chart types](./chart-types.md) — when to use each chart type

## Viewing your charts

1. Select **Charts** in the left menu.

You'll see all charts in your organisation. Use the search bar to find a chart by name.
