# Exporting and deleting

## Exporting a report as PDF

1. Open the report you want to export.
2. Select the **download icon** in the top-right corner.
3. The report downloads as a PDF file to your device.

## Deleting a report

1. On the reports list, select the **⋯ menu** on the report row.
2. Select **Delete**.

:::warning
Deleting a report is permanent. The report and all its comments will be removed and cannot be recovered.
:::
