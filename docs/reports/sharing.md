# Sharing reports

**Share a report with anyone — including people outside your organisation — via a public link or by email.**

## Sharing options

1. Select the **share icon** (↗) on the report row in the **Actions** column, or in the top-right corner when viewing a report.

![Share menu](/img/reports/reports_share.png)

2. Choose one of the options:

### Share via link

Generates a public URL that anyone with the link can use to view the report — no Dalgo account required.

- Toggle sharing **on** to activate the link.
- Toggle it **off** to deactivate the link (the URL will stop working).
- The dialog shows how many times the link has been accessed.

### Embed in email

Sends the report as a PDF attachment to one or more email addresses.

1. Enter recipient email addresses, separated by commas.
2. Optionally add a custom subject line.
3. Select **Send**.
