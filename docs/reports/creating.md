# Creating a report

**Create a report from any existing dashboard to capture its charts filtered to a specific date range.**

## Creating a new report

1. Select **Reports** in the left menu.
2. Select **+ CREATE REPORT** in the top-right corner.

![Create report dialog](/img/reports/reports_create.png)

3. Choose the **Dashboard** you want to base the report on.
4. Give your report a **Name** (e.g. "Q1 2025 Program Report").
5. Select the **Date column** Dalgo should use to filter data by date range.
6. Set the **Start date** and **End date** for the period you want to cover.
7. Select **Create**.

:::info
The report captures your dashboard's layout and chart settings at the time of creation. If you later change the original dashboard, existing reports are not affected.
:::

## Viewing a report

1. Select any report row in the list to open it.

![Report detail view](/img/reports/reports_detail.png)

2. At the top you'll see the report's **title**, **date range**, **creator**, and a link to the original **dashboard**.
3. The **Executive Summary** section appears at the top — a free-text area where you or your team can add notes and key takeaways.
4. Below the summary, all charts from the dashboard are shown, filtered to the report's date range.
