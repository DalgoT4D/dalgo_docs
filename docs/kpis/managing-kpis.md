# Managing KPIs

Once you've created KPIs, you'll need to monitor performance, organize views, and maintain targets over time. This guide covers the ongoing management of your KPI program.

## KPI Dashboard Overview

The main KPIs page provides a comprehensive view of organizational performance with:

- **Visual status indicators**: Color-coded cards showing at-a-glance performance
- **Trend charts**: Historical performance for each KPI
- **Filtering and search**: Focus on specific KPIs by status, type, or program
- **Performance metrics**: Current values, targets, and period-over-period changes

## Viewing and Monitoring KPIs

### Card Layout

Each KPI displays as a visual card containing:

**Header**
- KPI name and optional subtitle (program tags)
- Current status indicator (🟢/🟡/🔴)

**Main Value**
- Current period value
- Target value for comparison
- Period-over-period change percentage

**Trend Chart**
- Historical performance over recent periods
- Visual representation of the underlying metric data
- Contextual information about data freshness

**Quick Actions Menu**
- View detailed KPI information
- Edit KPI configuration  
- Delete KPI (if no longer needed)

### Status Interpretation

**🟢 Green - On Track**
- Performance meets or exceeds your green threshold
- Continue current strategies and monitor regularly
- Good candidates for showcasing success

**🟡 Amber - Needs Attention**  
- Performance is below target but within acceptable range
- Investigate causes and consider corrective actions
- Monitor more closely for further decline

**🔴 Red - Off Track**
- Performance is significantly below expectations
- Requires immediate investigation and intervention
- Priority focus for management attention

## Filtering and Search

### Search by Name
- Use the search box to find specific KPIs
- Search works on KPI names and descriptions
- Results filter as you type

### Filter by Status
Focus your attention where needed:
- **All Statuses**: View complete KPI portfolio
- **On Track**: See KPIs meeting targets  
- **Needs Attention**: Focus on amber-status KPIs
- **Off Track**: Priority KPIs requiring intervention

### Filter by Type
Organize by role in your theory of change:
- **Input**: Resource and investment KPIs
- **Output**: Activity and delivery KPIs  
- **Outcome**: Short-term result KPIs
- **Impact**: Long-term effect KPIs

### Filter by Program
View KPIs by organizational structure:
- Department or team KPIs
- Project or initiative tracking
- Geographic region performance
- Funding source or grant metrics

### Combined Filtering
Use multiple filters simultaneously:
- "Education" program + "Off Track" status = Education KPIs needing attention
- "Output" type + "On Track" status = Successfully delivered activities

## Detailed KPI Views

### Opening KPI Details

Click any KPI card to open the detailed view showing:

**Performance Summary**
- Current value vs. target
- Historical trend chart with more periods
- Status indicator and threshold explanation

**Configuration Details**
- Underlying metric information
- Target and threshold settings
- Time grain and measurement frequency

**Usage Context**
- Program tags and categorization
- Creation and last update timestamps
- Links to related charts or dashboards

### Analyzing Performance

**Trend Analysis**
- Look for patterns over time
- Identify seasonal fluctuations
- Spot declining or improving performance

**Target Assessment**  
- Check if targets remain appropriate
- Consider if external factors are affecting performance
- Evaluate need for target adjustments

**Data Quality**
- Verify data freshness and completeness
- Check for unusual spikes or gaps
- Ensure calculations align with expectations

## Editing KPIs

### When to Edit KPIs

**Target Adjustments**
- Performance consistently above/below targets
- Changed business priorities or market conditions
- Learning from historical performance data

**Threshold Refinements**
- Status indicators don't align with business reality
- Too many false alarms or missed warnings
- Better understanding of normal performance variation

**Organization Updates**
- Program restructuring or renaming
- Changed categorization needs
- Updated descriptions or context

### Editing Process

1. **Open KPI details**: Click the KPI card
2. **Click Edit**: Access the configuration form
3. **Make changes**: Modify targets, thresholds, or organization
4. **Preview impact**: See how changes affect status calculations
5. **Save updates**: Apply changes to the KPI

### What You Can Edit

**Targets and Thresholds**
- Adjust target values based on new goals
- Modify green/amber threshold percentages
- Change direction (higher vs. lower is better)

**Organization**  
- Update program tags
- Change metric type categorization
- Modify name or description

**Time Settings**
- Adjust time grain (daily, weekly, monthly, etc.)
- Change time dimension column if needed

### What You Cannot Edit

- **Underlying metric**: The foundational calculation cannot be changed
- **Historical data**: Past performance values are preserved
- **Creation timestamp**: Original creation date remains fixed

## KPI Lifecycle Management

### Regular Review Schedule

**Weekly Reviews**
- Check red and amber status KPIs
- Investigate performance issues
- Take corrective actions where needed

**Monthly Assessments**
- Review all KPI performance trends
- Assess target appropriateness
- Update descriptions or context as needed

**Quarterly Planning**
- Align KPI targets with strategic objectives
- Add new KPIs for emerging priorities
- Archive KPIs no longer relevant to goals

### Target Management

**Performance-Based Adjustments**
- Increase targets when consistently exceeded
- Decrease targets if persistently unachievable
- Adjust for seasonal patterns or market changes

**Strategic Alignment**
- Update targets to reflect new organizational goals
- Align with revised budgets or resource allocation
- Incorporate stakeholder feedback and expectations

### Portfolio Optimization

**KPI Count Management**
- Aim for 5-15 KPIs per program area
- Focus on the most critical performance indicators
- Avoid tracking everything; prioritize vital few

**Balanced Scorecards**
- Include mix of input, output, outcome, and impact KPIs
- Balance leading and lagging indicators
- Ensure coverage of all key organizational objectives

## Creating Action Plans

### For Red Status KPIs

1. **Investigate root causes**: Why is performance off track?
2. **Identify interventions**: What can be done to improve?
3. **Set timelines**: When should improvements be visible?
4. **Assign accountability**: Who will lead improvement efforts?
5. **Monitor closely**: Increase review frequency

### For Amber Status KPIs  

1. **Analyze trends**: Is performance improving or declining?
2. **Check external factors**: Are there temporary influences?
3. **Consider adjustments**: Are targets or processes realistic?
4. **Plan monitoring**: How closely to watch going forward?

### For Green Status KPIs

1. **Understand success**: What's driving good performance?
2. **Share learnings**: How can other areas benefit?
3. **Maintain performance**: What's needed to sustain success?
4. **Consider stretch goals**: Should targets be more ambitious?

## Best Practices

### Monitoring Frequency

- **Daily**: Operational KPIs requiring immediate attention
- **Weekly**: Team performance and service delivery metrics
- **Monthly**: Financial and program outcome indicators
- **Quarterly**: Strategic objectives and impact measures

### Communication

- Share KPI performance in regular team meetings
- Include status updates in stakeholder reports
- Celebrate achievements when targets are exceeded
- Transparently discuss challenges and improvement plans

### Continuous Improvement

- Regularly review threshold appropriateness
- Adjust targets based on organizational learning
- Retire KPIs that no longer provide value
- Add new KPIs as priorities evolve
