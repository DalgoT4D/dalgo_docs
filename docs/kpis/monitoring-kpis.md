# Monitoring KPIs

**Use the KPI dashboard to track performance, understand trends, and identify areas needing attention.**

The KPI dashboard provides an overview of all your Key Performance Indicators with visual cards showing current status, trends, and progress toward targets.

## KPI dashboard overview

The main KPIs page displays all your KPIs as cards with:

## Reading KPI cards

### Status indicators
Each KPI card shows a colored dot indicating performance:

- **🟢 Green**: On track - performing at or above green threshold
- **🟠 Amber**: Needs attention - between amber and green thresholds  
- **🔴 Red**: Off track - below amber threshold
- **⚫ Gray**: No data available for current period

### Trend arrows
Arrows show the direction and magnitude of change from the previous period:
- **⬆️ Green up arrow**: Improvement in desired direction
- **⬇️ Red down arrow**: Decline in desired direction
- **➡️ Gray horizontal**: No significant change
- **—**: No previous period data available

### Value display
- **Current value**: Latest calculated value for the current time period
- **Target value**: Your defined goal
- **Percentage**: Current value as percentage of target (when above/below 100%)

## Filtering and organizing

### Filter by status
Use the status filter to focus on KPIs needing attention:
- **All**: Show all KPIs
- **On Track**: Only green status KPIs
- **Needs Attention**: Only amber status KPIs
- **Off Track**: Only red status KPIs

### Filter by metric type
Filter by the type of metric being tracked:
- **Input**: Resource and investment metrics
- **Output**: Direct product and service metrics
- **Outcome**: Short-term result metrics
- **Impact**: Long-term change metrics

### Filter by program tags
If you've tagged KPIs by program area, filter to see specific program performance.

### Search
Use the search bar to find specific KPIs by name or metric.

### Data table
The table shows:
- **Period**: Time period label (e.g., "Q3 2024", "March 2024")

- **Change**: Difference from previous period
- **% Change**: Percentage change from previous period

## Taking action on KPIs

### When KPIs are off track (Red)
1. **Review recent annotations** to understand context
2. **Check data quality** - is the underlying data complete?
3. **Analyze contributing factors** - what changed in your program?
4. **Plan interventions** to improve performance
5. **Add annotations** documenting your response

### When KPIs need attention (Amber)
1. **Monitor closely** for signs of improvement or decline
2. **Review trend direction** - improving or worsening?
3. **Consider early interventions** to prevent further decline
4. **Document context** if external factors are involved

### When KPIs are on track (Green)
1. **Maintain current approaches** that are working
2. **Document successful practices** in annotations
3. **Consider** if targets should be adjusted upward
4. **Share lessons learned** with other program areas

## Best practices

### Regular monitoring
- **Weekly reviews**: Quick status check on all KPIs
- **Monthly analysis**: Deeper dive into trends and changes
- **Quarterly planning**: Adjust targets and strategies based on performance

### Using annotations
- **Document context** when values change significantly
- **Explain external factors** that impact performance
- **Record interventions** and their expected impact
- **Add beneficiary feedback** to provide human context

### Data-driven decisions
- **Combine KPI data** with other information sources
- **Look for patterns** across multiple related KPIs
- **Consider leading indicators** that predict future performance
- **Balance quantitative data** with qualitative insights
