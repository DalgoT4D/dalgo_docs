# Getting help

**The fastest way to get help is Discord. For bugs and account issues, email is best.**

## Discord — quick questions

Join the [Dalgo Discord server](https://discord.gg/C7BKvYPufp) and post in the **#dalgo-support** channel. The Dalgo team and the community check it regularly.

Best for:
- "How do I do X?"
- "Is this a bug or expected behaviour?"
- Questions while you're actively setting something up

## Email — bugs and account issues

Email **support@dalgo.org** for:
- Bugs that affect your data or pipelines
- Billing and subscription changes
- Account access issues (locked out, invitation not received)
- Requesting a new feature

When reporting a bug, include:
1. What you were trying to do
2. What happened instead
3. Any error message you saw (a screenshot helps)
4. The approximate time it happened

## GitHub — open-source contributions

Browse or open issues at [github.com/DalgoT4D](https://github.com/DalgoT4D). If you're technical and want to contribute a fix or feature, pull requests are welcome.
