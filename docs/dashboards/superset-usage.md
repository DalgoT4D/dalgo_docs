# Superset Usage

**The Superset Usage dashboard shows how your organisation is using Dalgo — which dashboards are being viewed, by whom, and how often.**

:::note
The Superset Usage dashboard requires a **Dalgo + Superset** subscription. If you see an error when opening this page, your organisation may not have Superset enabled. Contact support@dalgo.org for more information.
:::

## Accessing the Usage dashboard

1. Select **Dashboards** in the left menu.
2. Select **Superset Usage**. The dashboard loads automatically as an embedded Superset view.

![Usage Dashboard](/img/managedata/usage_dashboard.png)

## Metrics tab

The **Metrics** tab shows:

- **Active Users** — a bar graph of active and inactive users. A user is counted as active if they have visited at least one dashboard.
- **List of users** — all users in your organisation with their total dashboard visits.
- **List of dashboards** — all dashboards with their total visit count.

Filters at the top let you view metrics by **month**, **role**, and **dashboard**.

## Trends tab

The **Trends** tab shows:

- **Number of users accessing dashboards** — a monthly trendline of unique dashboard viewers, with change vs. the previous month.
- **Visits per user** — a monthly trendline of the average visits per user, with change vs. the previous month.

Top-level filters let you slice trends by role and dashboard.
