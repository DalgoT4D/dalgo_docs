# Platform Usage Metrics & KPIs

**Track key performance indicators (KPIs) for your Dalgo platform adoption and engagement. Monitor which dashboards are being viewed, by whom, and how often to measure the impact of your data initiatives.**

:::note
The Superset Usage dashboard requires a **Dalgo + Superset** subscription. If you see an error when opening this page, your organisation may not have Superset enabled. Contact support@dalgo.org for more information.
:::

## Accessing Platform Metrics

1. Select **Dashboards** in the left menu.
2. Select **Superset Usage**. The dashboard loads automatically as an embedded Superset view.

This dashboard provides pre-built KPIs and metrics to help you understand platform adoption, user engagement, and content utilization across your organization.

![Usage Dashboard](/img/managedata/usage_dashboard.png)

## Key Performance Indicators (Metrics Tab)

The **Metrics** tab displays essential KPIs for platform monitoring:

### User Engagement KPIs
- **Active Users** — visual comparison of active vs. inactive users. A user is counted as active if they have visited at least one dashboard within the selected time period.
- **User Activity Summary** — complete list of all users with their total dashboard visits, helping identify power users and those who may need additional training.

### Content Performance KPIs
- **Dashboard Utilization** — ranked list of all dashboards by total visit count, revealing which content drives the most value.

### Advanced Filtering
Apply filters by **month**, **role**, and **dashboard** to analyze KPIs across different dimensions:
- Track monthly performance trends
- Compare engagement across user roles
- Measure individual dashboard performance

## Trend Analysis & Growth Metrics

The **Trends** tab provides time-series KPIs for tracking platform growth and adoption:

### Growth & Adoption KPIs
- **User Growth** — monthly trendline showing unique dashboard viewers with month-over-month change indicators, helping measure platform adoption velocity.
- **Engagement Intensity** — monthly trendline of average visits per user with percentage change, indicating whether users are becoming more or less engaged over time.

### Strategic Insights
Use these trend KPIs to:
- **Measure adoption success** — track if more users are accessing the platform
- **Monitor engagement health** — ensure existing users remain active
- **Identify usage patterns** — spot seasonal trends or impact of training initiatives
- **Plan capacity and support** — anticipate resource needs based on growth trends

### Segmented Analysis
Apply top-level filters to analyze trends by:
- **User role** — compare adoption patterns across different organizational functions
- **Dashboard type** — identify which content categories drive sustained engagement

## Using Metrics as Organizational KPIs

Transform these platform metrics into actionable organizational KPIs:

### Data Culture KPIs
- **Monthly Active Users %** — percentage of licensed users actively using dashboards
- **User Engagement Score** — average visits per active user per month
- **Content Utilization Rate** — percentage of dashboards receiving regular views

### Training & Adoption KPIs
- **New User Onboarding Success** — time from user creation to first dashboard view
- **Role-based Adoption** — engagement rates by department or function
- **Dashboard Content Performance** — identify high-value vs. under-utilized content

### Success Benchmarks
Establish targets based on organizational goals:
- **Adoption Target**: 80% of users active monthly
- **Engagement Target**: 15+ dashboard views per active user per month
- **Content Efficiency**: Top 20% of dashboards drive 80% of views
