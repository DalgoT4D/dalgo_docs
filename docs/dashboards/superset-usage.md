# KPIs and Metrics: Superset Usage

**Track key performance indicators (KPIs) and analytics metrics for your organisation's Dalgo usage — monitor dashboard engagement, user activity, and platform adoption trends.**

:::note
The Superset Usage dashboard requires a **Dalgo + Superset** subscription. If you see an error when opening this page, your organisation may not have Superset enabled. Contact support@dalgo.org for more information.
:::

## Accessing the Usage dashboard

1. Select **Dashboards** in the left menu.
2. Select **Superset Usage**. The dashboard loads automatically as an embedded Superset view.

![Usage Dashboard](/img/managedata/usage_dashboard.png)

## Key Performance Indicators (Metrics tab)

The **Metrics** tab provides essential KPIs for monitoring platform engagement:

### User Engagement KPIs
- **Active Users** — Interactive bar graph showing active vs. inactive users over time. A user is classified as active if they have accessed at least one dashboard in the selected period.
- **User Activity List** — Complete roster of organisation users with their individual dashboard visit counts, enabling identification of power users and engagement patterns.

### Content Performance KPIs  
- **Dashboard Popularity** — Ranked list of all dashboards with total visit counts, helping identify most valuable content and underutilized resources.

### Filtering and Segmentation
Advanced filters enable granular analysis by:
- **Time Period** — Monthly, quarterly, or custom date ranges
- **User Role** — Segment metrics by different user types (Admin, Viewer, Editor, etc.)
- **Specific Dashboard** — Focus analysis on particular dashboard performance

## Analytics and Trends

The **Trends** tab delivers time-series analytics for strategic insights:

### Growth Metrics
- **User Adoption Trends** — Monthly trendline showing unique dashboard viewers with month-over-month growth percentages, tracking platform adoption and user base expansion.
- **Engagement Intensity** — Average visits per user over time with period-over-period change indicators, measuring how deeply users engage with the platform.

### Comparative Analytics
- **Period-over-period comparisons** — Automatic calculation of growth rates and percentage changes to identify trends and patterns.
- **Seasonal patterns** — Identify usage peaks, valleys, and cyclical behaviors in your organisation's analytics consumption.

### Advanced Filtering
Cross-dimensional analysis capabilities:
- **Role-based trends** — Compare adoption patterns across different user types
- **Dashboard-specific trends** — Track individual dashboard performance over time
- **Custom time ranges** — Flexible date selection for targeted analysis periods

## Using KPIs for Decision Making

### Monitor Platform Health
- Track active user ratios to ensure platform adoption
- Identify declining engagement early through trend analysis
- Measure ROI of dashboard investments through usage metrics

### Optimize Content Strategy
- Focus development efforts on high-traffic dashboard patterns
- Retire or improve underperforming dashboards
- Understand which content resonates with different user roles
