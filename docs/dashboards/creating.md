# Creating a dashboard

**Build a new dashboard by combining existing charts into a single view.**

## Creating a new dashboard

1. Select **Dashboards** in the left menu.
2. Select **+ Create Dashboard**. Dalgo immediately creates a new empty dashboard and opens the builder.
3. In the builder, select **+ Add Chart** to browse and add existing charts to the canvas.
4. Drag charts to reposition them. Drag the edges or corners to resize.
5. Select **Save** when you are done.

![Dashboard builder](/img/analysis/dashboard_builder.png)

:::info
Dalgo creates a draft dashboard the moment you select **+ Create Dashboard**. If you leave without saving, the draft is kept — you can continue editing it from the dashboard list.
:::

## Editing a dashboard

1. Select **Dashboards** in the left menu.
2. Open the dashboard you want to edit.
3. Select **Edit** (top-right of the dashboard).
4. Add, remove, or rearrange charts as needed.
5. Select **Save** when you are done.

## Setting a dashboard as your home page

Any dashboard can be set as your personal landing screen, shown when you open the **Impact** home.

1. Open the dashboard you want to use.
2. Select **Set as my landing page** from the dashboard menu (top-right).

:::note
This setting is personal — it only affects your account. Other users keep their own landing page.
:::
