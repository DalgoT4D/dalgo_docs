# Superset

**Superset is an optional hosted dashboarding environment within Dalgo for organisations that need more flexible, role-based visualisations.**

:::note
Superset is only available to organisations on the **Dalgo + Superset** plan. Contact support@dalgo.org to add it to your subscription.
:::

## Accessing Superset

1. Select **Dashboards** in the left menu.
2. Select the **Open Superset** option. A sign-in prompt will appear.
3. Select **Sign in with Google**.
4. If your email has been granted access, you will be logged in and Superset will open.

:::info Screenshot coming soon
A screenshot of the Superset sign-in prompt will be added here.
:::

Once logged in, your warehouse data is already connected. You can immediately start building charts, dashboards, and applying role-based access controls inside Superset.

:::note
If your email has not been granted access to Superset, contact your organisation's Superset administrator or reach out to support@dalgo.org.
:::

## Superset documentation

For help building charts and dashboards inside Superset, refer to the [official Superset documentation](https://superset.apache.org/docs/intro). Dalgo does not duplicate Superset's own docs.
