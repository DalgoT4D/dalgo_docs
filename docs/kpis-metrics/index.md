# KPIs & Metrics

Track progress toward your goals with reusable metrics and scannable KPI dashboards.

## What are Metrics and KPIs?

**Metrics** are saved calculations that you can reuse across charts, dashboards, and KPIs. Instead of defining the same calculation multiple times, create it once and reference it everywhere.

**KPIs** (Key Performance Indicators) are metrics with targets, status tracking, and trend monitoring. They give leadership a quick view of where the organization stands on its most important goals.

## Key Features

### 📊 **Reusable Metrics**
- Define calculations once, use everywhere
- Simple column aggregations or custom SQL expressions  
- Live preview and validation when creating
- Track which charts and KPIs use each metric

### 🎯 **Goal-Oriented KPIs** 
- Set targets with "higher is better" or "lower is better" direction
- Automatic RAG status: **On Track** (green), **At Risk** (amber), **Off Track** (red)
- Period-over-period change tracking
- Customizable status thresholds

### 📈 **Rich Visualizations**
- Trend charts with target lines
- Sparklines on KPI cards
- Dashboard widgets for executive views
- Timeline annotations with context

### 📝 **Context & Annotations**
- Add comments explaining KPI changes
- Capture beneficiary quotes with attribution
- Timeline view of all annotations
- Automatic value snapshots when adding entries

## Common Use Cases

### Program Tracking
Monitor beneficiary enrollment, completion rates, and outcome indicators with clear targets and RAG status.

### Financial Oversight  
Track revenue per beneficiary, cost efficiency, and budget utilization with trend analysis.

### Impact Measurement
Create KPIs for short-term outcomes and long-term impact indicators with beneficiary story integration.

### Executive Reporting
Build scannable dashboards that show organizational health at a glance.

## Best Practices

- **Start simple**: Begin with basic count or sum metrics before building complex calculations
- **Set realistic targets**: Base thresholds on historical performance and organizational capacity  
- **Add context**: Use annotations to explain significant changes or provide beneficiary perspectives
- **Review regularly**: Check KPI status weekly or monthly depending on your reporting cycle
- **Tag meaningfully**: Use program and indicator type tags to organize and filter KPIs
