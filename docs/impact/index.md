# Impact

**Impact at a Glance is the first screen you see when you log in — your personal home screen showing the dashboard your team has set as the default view.**

## The home screen

When you open Dalgo, you land here automatically. If a default dashboard has been configured for your account, it loads straight away and you can read charts, apply filters, and explore data without navigating elsewhere.

![Impact at a Glance home screen](/img/impact/impact_home.png)

If no landing page has been set yet, you'll see two options:

- **Choose a dashboard** — select an existing dashboard to display here
- **Create new dashboard** — build a new dashboard from scratch and set it as your home

## Setting your landing page

1. Select **Dashboards** in the left menu.
2. Open the dashboard you want to use as your home screen.
3. Select **Set as my landing page** from the dashboard menu (top-right corner).

The next time you log in, that dashboard appears on the Impact screen automatically.

:::info
Each user sets their own landing page independently. Changing yours has no effect on what other team members see when they log in.
:::
