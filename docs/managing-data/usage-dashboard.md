# Usage Dashboard

**The Usage Dashboard shows how your organisation is using the Dalgo platform — including which dashboards are being viewed, by whom, and how often.**

:::note
The Usage Dashboard requires a Superset subscription. If you see an error message when opening this page, your organisation may not have Superset enabled. Contact support@dalgo.org for more information.
:::

## Accessing the Usage Dashboard

1. Select **Usage Dashboard** under **Managing Data** on the left menu panel.

The dashboard loads automatically as an embedded Superset view.

![Usage Dashboard](/img/managedata/usage_dashboard.png)

## Metrics tab

The **Metrics** tab shows:

1. **Active Users** — a bar graph showing the total number of active and inactive users. A user is counted as active if they have visited at least one dashboard.
2. **List of users** — a table of all users in your organisation with their total dashboard visits.
3. **List of dashboards** — a table of all dashboards in your organisation with their total visit count.

Filters at the top let you view metrics by **month**, **role**, and **dashboard**.

## Trends tab

The **Trends** tab shows:

1. **Number of users accessing dashboards** — a monthly trendline of how many users are viewing dashboards, with the change compared to the previous month.
2. **Visits per user** — a monthly trendline of the average number of visits per user, with the change compared to the previous month.

Top-level filters allow you to slice trends by role and dashboard.
