# The Data Pipeline Process

Dalgo's data pipeline consists of four main processes:

1. **[Ingest](process/ingest):** Connect your data sources to a single warehouse.
2. **[Transform](process/transform):** Apply data transformations using DBT and UI4T (UI based transformation).
3. **[Visualization](process/visualization):** Create and view dashboards in Superset.
4. **[Orchestration](process/orchestration):** Schedule and automate your data pipelines.
