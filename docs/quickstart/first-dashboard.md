# Viewing your first dashboard

**Dashboards are collections of charts. This step shows you how to open one and understand what you're looking at.**

## Steps

1. Select **Dashboards** in the left menu. You'll see all dashboards in your organisation.
2. Select any dashboard name to open it.
3. Each tile on the dashboard is a chart. Hover over a chart to see its exact values.
4. If the dashboard has filters at the top, use them to narrow the data by date, region, or other dimensions.

:::info
Dashboards always show the latest data — they update automatically each time your data pipeline runs.
:::

## Can't see any dashboards?

If the dashboard list is empty, your implementation partner may not have created any yet. Reach out to them, or contact support@dalgo.org.
