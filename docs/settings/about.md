# About

**The About page shows your platform version, organisation details, and links to open-source licence information.**

:::info Screenshot coming soon
Screenshots for the About page will be added in a future update.
:::

## Accessing About

1. Select **Settings** in the left menu.
2. Select **About**.

## What you'll see

- **Platform version** — the version of Dalgo your organisation is running
- **Organisation name** — your registered organisation name
- **Licence information** — Dalgo is open source under the [AGPL-3.0 licence](https://github.com/DalgoT4D/DDP_backend/blob/main/LICENSE)
