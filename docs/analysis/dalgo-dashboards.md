# Dalgo Native Dashboards

**Dalgo's built-in dashboards let you build charts from your data and arrange them into shareable views — no separate BI tool required.**

You can create multiple dashboards for different audiences, such as a field team view and a donor report.

## Viewing your dashboards

1. Select **Dashboards** on the left menu panel.

![Dashboard list](/img/analysis/dashboard_list.png)

You will see all dashboards in your organisation. Use the search bar to find a dashboard by name.

## Creating a dashboard

1. Select **+ Create Dashboard**. Dalgo immediately creates a new empty dashboard and opens the dashboard builder.
2. In the builder, select **+ Add Chart** to browse and add existing charts to the canvas.
3. Drag charts to reposition them. Drag the edges or corners to resize.
4. Select **Save** when you are done.

![Dashboard builder](/img/analysis/dashboard_builder.png)

:::info
Dalgo creates a draft dashboard the moment you select **+ Create Dashboard**. If you leave without saving, the draft is kept — you can continue editing it from the dashboard list.
:::

## Setting a dashboard as your home page

Any dashboard can be set as the first screen you see when you log in.

1. Open the dashboard you want to use.
2. Select **Set as my landing page** from the dashboard menu.

The dashboard will now appear on your **Impact at a Glance** home screen every time you log in.

## Charts

Charts are created separately and then added to dashboards.

### Creating a chart

1. Select **Charts** on the left menu panel.
2. Select **+ New Chart**.
3. Choose a chart type:
   - **Bar Chart** — compare values across categories
   - **Pie Chart** — show proportions of a whole
   - **Line Chart** — display trends over time
   - **Number** — show a single metric value
   - **Map** — visualise data by geography
   - **Table** — display raw data in rows and columns
4. Select the **dataset** (database table) your chart will draw from, then select **Configure**.

![Chart type selector](/img/analysis/chart_type_selector.png)

5. In the configuration screen, choose which columns to use for axes, values, filters, and labels.
6. Give your chart a name and select **Save**.

The chart is now available to add to any dashboard.

### Editing a chart

1. Select **Charts** on the left menu panel.
2. Find the chart you want to edit and select the **⋯ menu**.
3. Select **Edit**.
4. Make your changes and select **Save**.
