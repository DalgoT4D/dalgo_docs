# Dalgo Native Dashboards

**Dalgo native dashboards are built directly into the platform for routine reporting and monitoring without requiring a separate BI tool.**

## What you can do

- View dashboards directly within Dalgo.
- Track operational and program metrics from the data already flowing through your pipelines.
- Use the built-in dashboarding experience for standard reporting needs.

## When to use Dalgo native dashboards

Dalgo native dashboards are a good fit when you want a simpler, built-in visualization experience inside Dalgo itself. If your team needs more advanced customization or role-based access control for dashboards, use [Superset](./superset).
