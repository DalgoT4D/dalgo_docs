# Superset

**Superset is available as an optional hosted dashboarding environment within Dalgo for organisations that need more flexible and role-based visualisations.**

:::note
Superset is only available if your organisation has subscribed to the Dalgo + Superset plan. Contact support@dalgo.org if you want to add it to your subscription.
:::

## Accessing Superset

1. Select **Dashboards** on the left menu panel.
2. If Superset is enabled for your organisation, select the **Open Superset** option. A sign-in prompt will appear.
3. Select **Sign in with Google**.
4. If your email has been granted access, you will be logged in and Superset will open.

<!-- SCREENSHOT: Superset sign-in prompt inside Dalgo -->

Once logged in, your warehouse data is already connected. You can immediately start building charts, dashboards, and applying role-based access controls inside Superset.

:::note
If your email has not been granted access to Superset, contact your organisation's Superset administrator or reach out to support@dalgo.org.
:::
