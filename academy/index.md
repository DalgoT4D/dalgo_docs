# Welcome to Dalgo Academy

Dalgo consultants typically guide organizations on specific implementation tasks. Over time, we want every organization adopting Dalgo to become more self-sufficient and confidently use the platform's full capabilities.

Dalgo Academy is built for that purpose.

## How Dalgo Academy is Structured

Dalgo Academy has two parts:

1. [Foundations](/academy/Foundations/): Core concepts, setup guidance, and architecture decisions. This helps teams build the right mental model before implementation.
2. [Playbooks](/academy/Playbooks/): Practical implementation resources, including videos, data dictionary documentation practices, and best practices for tools like Superset and AI-assisted work with dbt, SQL, and Python.

In short, **Foundations** explains the _why/what for setup and architecture_, while **Playbooks** focuses on the _how_ and efficiency of day-to-day execution.
